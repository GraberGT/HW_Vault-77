В программе должен быть реализован метод, который в качестве аргументов принимает код товара (например 'А01') и сумму, которую нужно за этот товар заплатить. 

Условия работы метода:

Вернуть название товара, если введенная сумма совпадает с ценой товара.
Вернуть название товара и сумму сдачи, если введенная сумма превышает цену товара.
Вернуть сообщение "Внесите еще #{недостающая сумма}", если введенной суммы недостаточно для покупки товара.
Вернуть сообщение "#{название товара}: закончился", если такого товара больше нет.
 

При этом, при каждой покупке товара, его количество (quantity в хэше для каждого товара) должно уменьшаться. Количество денег в машине не ограничено.

 

Пример имеющихся в машине товаров:

items = [{:name=>"Snickers", :code=>"A01", :quantity=>10, :price=>250},

         {:name=>"Pepsi", :code=>"A02", :quantity=>5, :price=>350},

         {:name=>"Orange Juice", :code=>"A03", :quantity=>10, :price=>400},

         {:name=>"Bon Aqua", :code=>"A04", :quantity=>7, :price=>120}, 

         {:name=>"Bounty", :code=>"A05", :quantity=>10, :price=>270}]

 

Использовать этот массив хэшей-товаров в программе как изначальный набор товаров в автомате.
