1: Задание 1
С этого урока мы будем создавать объектную модель (классы и методы) для гипотетического приложения управления железнодорожными станциями, которое позволит управлять станциями, принимать и отправлять поезда, показывать информацию о них и т.д.

 

Требуется написать следующие классы:

 

Класс Station (Станция):

Имеет название, которое указывается при ее создании (при создании объекта класса Station).
 

Класс Route (Маршрут):

При создании принимает два аргумента: начальную и конечную станции (объекты класса Station).
Может возвращать список станций в маршруте (пока только две станции)
Имеет название, состоящее из названий станций.
 

Класс Train (Поезд):

Имеет номер (произвольная строка), тип (грузовой, пассажирский) и количество вагонов, эти данные указываются при создании экземпляра класса
Может набирать скорость
Может возвращать текущую скорость
Может тормозить (сбрасывать скорость до нуля)
Может возвращать количество вагонов
Может прицеплять/отцеплять вагоны (по одному вагону за операцию, метод просто увеличивает или уменьшает количество вагонов). Прицепка/отцепка вагонов может осуществляться только если поезд не движется.
