1. Создать хэш, содержащий месяцы и количество дней в месяце (месяц – ключ, кол-во дней – значение). Перебрать хэш и в цикле выводить только те месяцы, у которых кол-во дней равно 30.

2. Написать программу, которая заполнит массив числами от 10 до 100 с шагом 5. Пример массива: [10, 15, 20, 25, ..., 100]

3. Написать программу, которая заполнит массив числами Фибоначчи до 100. Пример массива: [0, 1, 1, 2, 3, 5, 8, ...]

4. Написать программу, которая создаст и заполнит хэш гласными буквами, где значением будет являться порядковый номер буквы в алфавите (например: А - 1).
