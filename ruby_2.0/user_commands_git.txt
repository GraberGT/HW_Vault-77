git init - создание репозитория в папке
git status - проверка работоспособности
git add - подготовка файла к коммиту
git log - просмотр изменений
git log -p - работа с историей изменений с помощью клавиш f и b
git show - изменения в конкретном коммите
git branch new-feature - создание новой ветки
git checkout название ветки - переход на другую ветку
git merge выполняется из ветки, в которую мы сливаем изменения, а аргументом передаётся имя ветки, которую мы сливаем.
git push - отправка файла на удаленный репозиторий
