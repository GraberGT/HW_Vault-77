Написать программу-ассистент, которая будет совершать заданные действия по заданным инструкциям

 

Написать метод perform, который принимает два аргумента: строку, представляющую какое-то действие, и набор исходных данных (массив). А также метод принимает блок, в котором будет производиться то действие над теми исходными данными, которые передаются в метод.

 

Аргументы, которые должен принимать метод:

Действие: 'find words', исходные данные: массив строк, строки должны состоять из нескольких слов. Метод должен вернуть ту строку, в которой содержится наибольшее количество слов.
Действие: 'parse numbers', исходные данные: массив чисел. Метод должен преобразовать массив в хэш, в котором ключом будет являться порядковый номер числа (отсчет должен начинаться с единицы), а значением - само число, и вернуть его в качестве результата.
 

Все действия над исходными данными не должны производиться в методе, они должны выполняться в блоке. Метод должен лишь принять нужные данные (аргументы) и вернуть результат.

 

Если в метод не передается блок, он должен вернуть сообщение "Пожалуйста, дайте мне инструкции".

 

Подсказка: однострочные блоки принято записывать с помощью фигурных скобок { }, многострочные блоки - с помощью do ... end.

При передаче блока в метод, его можно записать разными способами, в зависимости от того, уместится ли код в блоке в одну строку.
